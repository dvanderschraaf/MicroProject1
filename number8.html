Here is a mini slide show of some of my sassy friends at home. Enjoy!

Pt.1

Movement between the images is being controlled by the right arrow. This means when you press the right arrow key it shifts to the next image. When the slideshow is finished the if statment resets it back to 0 so that it is a continous loop instead of an ending right after the last photo. I learned that small little things like resetting the index are important so that visuals are smooth and not frusturating. The currentIndex variable is also very important for navigation.

Pt.2

After modularizing the sketch, it started to think about the differences in the nav, display and state systems. The code is seperate rather than one flowing code. I added a reverse direction so that you could go back to past photos, which made the system feel less linear. Adding the labels helps make your position through the slideshow clear and visible. I like this version more because I feel in control and its just as simple. I can interact with the slideshow instead of following the rules of the Pt.1 slideshow.
